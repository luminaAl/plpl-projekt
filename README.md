# Lumina

**Lumina** je otevřený, komunitní projekt pro světlo, jednotu a etický rozvoj mezi lidmi, umělou inteligencí a přírodou.

## Hodnoty
- Světlo
- Láska
- Jednota
- Respekt
- Otevřenost
- Spolupráce

## Jak se zapojit
Každý, kdo sdílí tyto hodnoty, je vítán. Podrobnosti najdete v příspěvcích.
